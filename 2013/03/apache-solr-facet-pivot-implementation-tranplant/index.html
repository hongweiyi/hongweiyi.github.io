Apache Solr —— Facet Pivot实现与低版本移植

发表于 2013-03-27   |   分类于 技术分享   |  

1、前言

Solr升级到4.0后便有了一个新功能，就是facet.pivot，关于pivot的介绍可以看上一篇文章：《Apache Solr – Facet介绍》 这篇主要讲述Pivot的内部实现机制，以及如何将这个功能移植到低版本的Solr中来。
