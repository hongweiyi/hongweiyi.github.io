Python - 遍历目录

发表于 2011-12-21   |   分类于 技术分享   |  

Python遍历目录经常要用到，所以就在日志mark一下吧。

最开始是用传统的方式写的，找到目录，list它的所有下一级目录，再递归遍历，代码如下：

  1. import os, sys 
  2.  
  3. # list files in root by recursion  
  4. def listfile(root, callback = None):  
  5.     for path in os.listdir(root):  
  6.         abs_path = os.path.join(root, path)  
  7.         if os.path.isdir(abs_path):  
  8.             listfile(abs_path, callback)  
  9.         else:  
  10.             if callback: callback(abs_path)  
  11.  
  12. #do something to the path  
  13. def dosth(path):  
  14.     print path  
  15.  
  16. if name == "main":  
  17.     path = raw_input(‘enter your path: ‘)  
  18.     try:  
  19.         listfile(path, dosth)  
  20.     except: # Exception in permissions, sytem file etc  
  21.         pass 

看了上面的代码总觉得别扭，连轮子都有的python应该有遍历目录的方法吧，Google之后发现真有，即os.walk(path)方法。

os.walk()可以得到一个三元元组(dirpath, dirnames, filenames)，dirpath是字符串，为起始路径；dirnames是list，为dirpath下的文件夹名；filenames是list，为dirpath下的文件名。

其中dirnames以及filenames均不包含路径信息，如果需要得到全路径，需要使用os.path.join(path,name)。

  1. import os, sys  2.   3. # list files in root by os.walk()   4. def walkfile(root, callback = None):   5.     for dirpath, dirnames, filenames in os.walk(root):   6.         for filename in filenames:   7.             abs_path = os.path.join(dirpath, filename)   8.             if callback: callback(abs_path)   9.   10. #do something to the path   11. def dosth(path):   12.     print path   13.   14. if name == "main":   15.     path = raw_input(‘enter your path: ‘)   16.     #In theory, there is no permissions/systemfile exception by os.walk   17.     walkfile(path, dosth)
